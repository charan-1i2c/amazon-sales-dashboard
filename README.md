# 📊 Amazon Sales Report Dashboard

An interactive **Amazon Sales Dashboard** built using **Microsoft Power BI** to analyze sales performance across different countries, cities, markets, categories, and shipping modes.

---

## 📌 Project Overview

This dashboard provides a comprehensive overview of Amazon sales data through interactive visualizations and KPIs. Users can filter the report by **Category** and **Market** to gain deeper insights into sales performance.

---

## 🚀 Features

- 📈 Total Sales KPI
- 💰 Total Profit KPI
- 📦 Total Quantity Sold
- 🚚 Total Shipping Cost
- 🌍 Sales Analysis by Country
- 🏙️ Sales Analysis by City
- 📂 Sales Distribution by Category
- 🌎 Sales Distribution by Market
- 🚛 Sales by Shipping Mode
- 🎯 Interactive Slicers for Category and Market

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

## 📈 Key Insights

- Standard Class generated the highest sales.
- Technology contributed the largest share of overall sales.
- The United States recorded the highest sales among all countries.
- APAC was the highest-performing market.
- Sales can be dynamically filtered by product category and market.
